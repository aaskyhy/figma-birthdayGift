Birthday Surprise Interface
This project is a birthday gift that I designed and created for my best friend in 2025. It was made to celebrate our friendship in a unique and meaningful way through a personalized interactive interface.
It includes a custom interface, memorable messages, photos, and interactive elements to make the birthday surprise more special.

Tools Used: Figma (UI/UX Design)

The goal of this project was to improve my design skills while creating a thoughtful and memorable digital gift for someone important to me.
